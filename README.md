## <img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/bash/bash-original.svg" width="20" height="20"> The Terminal

Press **Ctrl+`** or click the **$_** button in the bottom-right corner to open the interactive terminal.

**Available Commands:**

```
help      - See all available commands
about     - Learn about my background
cv        - Get a link to download my CV
projects  - Browse my projects
skills    - See what I work with
contact   - Get my contact info
schedule  - Book a call with me
github    - Visit my GitHub profile
linkedin  - Check out my LinkedIn
clear     - Clear the terminal
exit      - Close the terminal
```

## <img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/github/github-original.svg" width="20" height="20"> How to Use This

### View It Live

The portfolio is hosted on GitHub Pages:

**[samir-guenchi.github.io/portfolio](https://samir-guenchi.github.io/portfolio)**

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Samir-Guenchi/portfolio.git
cd portfolio

# Open in your browser
# Windows:
start index.html
# macOS:
open index.html
# Linux:
xdg-open index.html
```

No build process, no npm install, no server needed.

## <img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/git/git-original.svg" width="20" height="20"> Want to Fork This?

This portfolio is open source under the MIT License.

**Steps to customize:**

1. Fork the repo
2. Clone your fork
3. Edit `index.html` with your content
4. Update colors and fonts in `style/style.css`
5. Modify terminal commands in `js/script.js`
6. Push to your GitHub
7. Enable GitHub Pages in Settings > Pages
